🚀 GitHub Pages 설정이 필요합니다

⚠️ 중요: 저장소 관리자가 GitHub Pages를 활성화해야 합니다.

서강대학교 우정원 식단표 서비스가 준비되었습니다!
아래 3단계만 거치면 바로 사용할 수 있습니다.

1 저장소 Settings로 이동

GitHub 저장소의 Settings 탭을 클릭하세요.

Settings 열기
2 Pages 메뉴 선택

좌측 사이드바에서 "Pages" 메뉴를 찾아 클릭하세요.

3 Source 설정

Build and deployment 섹션에서:

  • Source: "Deploy from a branch" 선택
  • Branch: main 선택
  • Folder: /docs 선택
  • Save 버튼 클릭!
✅ 배포 완료 후:
1-2분 후 이 페이지가 실제 서비스로 자동 전환됩니다!
